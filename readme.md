# Log parser app

Backend - MAYD/Apollon Assessment 

## Installation

```bash
$ npm install
```

## Running the app

```bash

# development
$ node parser.js --input ./app.log --output ./errors.json

# watch mode
$ npm run start:dev

```


## Environment

```bash
LOG_LEVEL = error
```
